< div class ="comment__item__main__hd" > < a class ="user-name" >景田饮用纯净水< / a >< / div > < p class ="" > 理解下。
1，首先肯定不存在涨，因为空间已经没有了，整个经济系统中，如果一部分经济体超出别的部分太多，会吸引太多资源，导致其他资源少，反过来，其他部分资源少，会引起这个强势部分的后继乏力，会引起整个系统的强烈不稳定。
例子参考木桶理论，现在是一块木太长了，其他木块显著都低，整个木桶承载能力不够，再涨木桶恐怕要漏水。
2， 跌怎么跌？
在这个木桶中，地产木块最长，其他木块都短，由于这个世界是一个整体，那么木桶能承载的水量其实是整个木桶总的体积决定的。
现在的可能是，房地产木块还那么长，但是其他木板因为这次疫情的原因，全部变短！ 这就恐怖了。
现在肯定不可能给资源给长木块了，因为给木桶加高的资源暂时就这么多，一边是地产在费精力维持不倒， 一边是因为疫情的影响，其他木块在大面积缩短。
导致整个木桶系统能承载的水量急剧下降！
所以，据现在的分析来看，我认为，政府会大规模给短木块修复，长木块能获得的维持资源会减少，所以我投票 2有大几率发生。
至少，也是个阴跌，或者快速阴跌， 因为整个木桶系统，短木块出现了系统性危机， 木桶外的备用木料（居民可支配收入） 已经不够了！ < / p > < !----> < / div >
